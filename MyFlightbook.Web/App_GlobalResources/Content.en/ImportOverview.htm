%APP_NAME% can import flights, whether from a spreadsheet that you create or as an export from other logbook programs. But regardless of the source, it must be in the form of a CSV (spreadsheet) file.

The first row of the table (spreadsheet) must be the set of headers that identify which column is which; each subsequent row of the spreadsheet represents a flight that you'd like to import.

If you're importing data that you've exported from another commercial logbook program, you may be able to import it directly; %APP_NAME% can translate the CSV exports from a number of commonly used logbook programs, including ForeFlight, LogTen Pro, Zululog, CrewTrac, CrewLog, and ELogSite.

You will have a chance to preview the flights and see any potential errors before anything is imported.
