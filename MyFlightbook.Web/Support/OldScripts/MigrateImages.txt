How to migrate aircraft images:

1) go into the aircraft directory and do dir *. /b > ac.txt
2) copy it to the clipboard and then into word
3) turn OFF smart quotes
3) delete ac.txt
4) in Visual Studio, replace "\n" with "', '", fix up the first/last items
5) Run this: 
    select Concat('xcopy /S /C /O /E /Y /I ', tailnumber, ' id\\', idaircraft) AS cmd
    from aircraft
    where tailnumber in (...)
  
6) Save the reslt as CSV
7) Open it up and copy to the clipboard
8) Run it.
9) can now sync change to use ID subdir.
